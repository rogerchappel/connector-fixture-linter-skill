# Contributing

Thanks for helping improve `connector-fixture-linter-skill`.

## Development

```bash
npm install
npm run release:check
```

Keep fixtures synthetic. Do not commit live connector payloads, tokens, customer records, or other sensitive data.

## Pull Requests

- Include a short description of the fixture or rule behavior being changed.
- Add or update tests for new lint rules.
- Run `npm run release:check` before requesting review.
